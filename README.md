# Roblox Exp Games

Games: Hide And Seek, Legends Of Speed and Impossible Squid game Obby. More coming out Soon!

Hide And Seek - Collect all Coins, Find All Players (Requires IT), Teleport to Safe Zone, Teleport to Danger Zone, Player WalkSpeed, Player JumpPower, Spectate Others, Player ESP

Legends Of Speed - Teleport all orbs to you, Teleport all Hoops to you.

Impossible Squid game Obby - Green and Red colored (glass) indicator that allows to see if its safe or danger
